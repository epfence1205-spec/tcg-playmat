# Keyboard Shortcuts

## Game Actions

| Key | Action | Description |
|-----|--------|-------------|
| `D` | Draw | Draw the top card from your Library into your Hand (Zone C) |

## Notes

- Keyboard shortcuts are disabled when focus is inside a text input, textarea, or contenteditable element (e.g., the deck import modal).
- All shortcuts work regardless of Caps Lock state (both `d` and `D` trigger Draw).
- The Draw shortcut has no effect when the Library is empty.
